# Doctor Appointments (MERN)

## Description
Web application to manage medical appointments with roles:
- Patient: book and view appointments
- Doctor: view agenda
- Admin Facility: manage facility appointments (confirm/cancel)

## Tech Stack
- Backend: Node.js, Express, MongoDB, JWT
- Frontend: React (Vite), React Router, Axios

## Project Structure
- backend/
- frontend/

## Requirements
- Node.js
- MongoDB
